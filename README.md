# OrderBankSlip
Aplicação que realiza a integração com a API da empresa Efí (Gerencianet) para a geração de boletos bancários. A aplicação oferece aos usuários a possibilidade de criar pedidos, especificando o valor, o nome e a quantidade dos produtos desejados. Em seguida, a aplicação gera um boleto bancário ou um QRCode, facilitando o pagamento do pedido.
